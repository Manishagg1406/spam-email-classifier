# Spam Email Classifier

A machine learning project to classify SMS/Email messages 
as Spam or Ham using NLP techniques.

## Dataset
UCI SMS Spam Collection Dataset

## Tech Stack
- Python
- NLTK (NLP preprocessing)
- Scikit-learn
- Naive Bayes + Logistic Regression
- Bag of Words + TF-IDF

## Models & Accuracy
| Model | Accuracy |
|-------|----------|
| NB + BoW | 97.76% |
| NB + TF-IDF | 96.50% |
| LR + BoW | 98.39% |
| LR + TF-IDF | 96.77% |

## Features
- Text preprocessing with NLTK
- Multiple model comparison
- Confusion matrix evaluation
- Interactive prediction tester

## How To Run
Open `spam_classifier.ipynb` in Google Colab or Jupyter Notebook
